# 基准线

一个参考 `fund.cc.cd` 思路做的实时基金看盘小工具，定位是“自选基金 + 持仓盈亏 + 提醒 + 按次付费服务”的产品原型。

## 运行

当前已启动本地服务：

```text
http://localhost:4177/
```

也可以直接打开 `index.html`。实时估值通过天天基金公开 JSONP 接口获取，浏览器能联网时会自动刷新。

## 服务器部署

- 当前远端运行目录：`/home/ubuntu/yangjibao`
- 后续 Git 开发目录建议：`/home/ubuntu/projects/jizhunxian`
- 后端端口：`8080`
- 本机检查：`curl http://127.0.0.1:8080/api/health`
- 启动方式：`cd ~/yangjibao && nohup python3 server.py > yangjibao.log 2>&1 &`
- 开机自启：已写入 `ubuntu` 用户的 `crontab @reboot`

如果公网无法访问 `http://服务器IP:8080/`，需要在云服务器安全组放行 TCP 8080，或放行 80/443 后再接 Nginx/域名。

## 已有功能

- 自选基金：输入 6 位基金代码，或用基金名称/拼音搜索后添加。
- 实时估值：展示估算净值、估算涨跌幅、上一交易日净值、估值时间。
- 自动刷新：默认每 60 秒刷新一次，也可以手动刷新。
- 排序：按自选顺序、涨跌幅、今日收益、名称排序。
- 持仓管理：录入每只基金持仓金额，自动估算今日盈亏。
- 智能提醒：本地保存涨跌幅提醒规则，并显示是否触发。
- Pro 页面：展示 AI 复盘、组合体检、策略回测、提醒点数等按次付费能力。
- 登录演示：本地保存演示账号和点数，未登录购买时会先拉起登录弹窗。
- 本地持久化：自选、持仓、提醒规则、演示账号保存在浏览器 `localStorage`。

## 建议收费功能

### 按次购买

- AI 收盘复盘：建议 ¥3/次，解释今日收益来源、主要拖累、明日关注。
- 组合体检报告：建议 ¥9/次，诊断行业暴露、基金重合度、回撤压力和调仓清单。
- 策略回测报告：建议 ¥19/次，验证补仓/止盈规则，输出胜率、最大回撤和 PDF 报告。
- 提醒点数包：建议 ¥12/30 条，按微信/短信触达次数扣点，点数不过期。

### 可升级的高阶服务

- 基金经理监控：经理变动、任期回撤、同类排名变化提醒。
- 重仓股异动归因：把基金涨跌拆到行业和前十大持仓。
- 基金对比器：费用、回撤、夏普、规模、换手率、持仓重合度。
- 云端同步：手机、电脑多端同步自选和持仓。

## 后续可接的真实后端

- 用户系统：手机号验证码、微信登录、第三方 Auth 或自建账号体系。
- 云端自选和持仓：替换当前 `localStorage`。
- 按次支付：微信支付、支付宝、Apple IAP，并保存订单和报告记录。
- 推送通道：公众号模板消息、企业微信、短信服务。
- 数据层：基金基础信息、净值历史、持仓、行业、指数行情、估值排名。

## 风险提示

实时估值为第三方公开接口估算值，仅供学习和产品原型演示，不构成投资建议。最终涨跌以基金公司披露净值为准。
